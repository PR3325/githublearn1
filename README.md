# Parasu Raman G Portfolio

This is a static portfolio website built with HTML, CSS, and JavaScript.

## Deploy on Netlify

1. Open https://www.netlify.com/
2. Sign in or create an account.
3. Click **Add new site**.
4. Choose **Deploy manually**.
5. Upload the `portfolio-deploy.zip` file, or drag the `portfolio-deploy` folder into Netlify.

No build command is needed.

## Edit Later

- Edit `index.html` to change text, projects, email, LinkedIn, GitHub, or resume link.
- Edit `styles.css` to change colors, spacing, and layout.
- Edit `script.js` to change menu or animation behavior.

After editing, upload the updated files to Netlify again.
