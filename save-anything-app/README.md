# SaveStack

SaveStack is a mobile-first prototype for the idea: store anything from social media or the web, organize it by category, and find it later without confusion.

## Current MVP

- Save any URL with title, platform, category, and note
- Filter by category
- Search saved items
- Favorite, edit, delete, and open saved links
- Data persists in browser localStorage
- Mobile-first responsive interface
- Light/dark theme toggle

## Next Build Phase

- Convert screens to React Native with Expo
- Add Firebase Authentication
- Store links in Cloud Firestore per user
- Add Android share target so users can share from YouTube, Instagram, browser, and other apps
- Add AI category suggestion and link summaries
